# P Sushanth | Professional Portfolio

A high-end, immersive digital experience showcasing my work in AI, Quantum Computing, and Full-Stack Development.

## ✨ Key Features
- **Advanced SVG Preloader**: A custom "S" logo with professional stroke-drawing animation.
- **Circular Ripple Transitions**: State-of-the-art theme toggling using the View Transition API.
- **High-Resolution Particle Background**: Interactive background that forms "P SUSHANTH" and reacts to mouse proximity.
- **Quantum Qubit Interaction**: A shimmering, 3D-effect "Qubit" in the Research section that vibrates and "entangles" when hovered.
- **Project Case Study Modals**: Deep-dive technical descriptions with glassmorphic UI.
- **Responsive & SEO Optimized**: Fully optimized for search engines and performance.

## 🛠️ Tech Stack
- **HTML5 / CSS3 / Vanilla JavaScript**
- **Inter & Newsreader Google Fonts**
- **Formspree** for contact handling
